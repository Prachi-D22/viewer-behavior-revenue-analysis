
# 📊 Impact of Viewer Behavior on Advertising and Subscription Revenue in Streaming Platforms

## 📌 Project Overview

This project analyzes how viewer behavior impacts advertising and subscription revenue in streaming platforms. It uses data analytics and machine learning techniques to identify patterns in user engagement and predict outcomes such as churn and revenue contribution.

---

## ❗ Problem Statement

Streaming platforms face challenges in understanding user behavior due to varying preferences and engagement levels. This makes it difficult to optimize subscription retention and advertisement revenue effectively.

---

## 💡 Proposed Solution

The system leverages machine learning to analyze user data and predict behavior patterns. The insights are visualized using Power BI dashboards to support data-driven decision-making.

---

## ⚙️ Technologies Used

* Python
* Power BI
* Pandas, NumPy
* Scikit-learn

---

## 🧠 Machine Learning Model

* Type: Classification Model
* Purpose: Predict user churn and engagement
* Features Used:

  * Watch time
  * Genre preference
  * Subscription type
  * User activity

---

## 📊 Dashboard (Power BI)

The Power BI dashboard provides:

* User behavior analysis
* Watch time trends
* Revenue insights
* Genre-based engagement

---

## 📈 Key Insights

* Higher watch time increases subscription retention
* Certain genres drive more engagement
* Active users contribute more to advertisement revenue
* ML model helps identify potential churn users
